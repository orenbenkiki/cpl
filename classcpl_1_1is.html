Detailed Description

template<typename T>
class cpl::is< T >

A holder of some value.

This allows creation of cpl::ptr and cpl::ref to the value. It is annoying we need to wrap the value in a class for this. Deriving from T makes it a bit easier to suffer, but also means that T can't be a primitive type.
